- Added Report Button (Client-Side) allowing users to flag inappropriate behavior.
- Implemented Custom Tooltip UI for better interaction and clarity.
- Introduced Improved User Presence System to handle real-time join and leave events accurately.
- Improved overall responsiveness and UI performance for a smoother user experience.
